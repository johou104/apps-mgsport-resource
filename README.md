# apps-mgsport-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面作为资源文件存储，不服务于任何特定域名或网站，仅供索引、查阅或离线使用。

## 目录结构

```
apps-mgsport-resource/
├── pages/        # 存放 HTML 页面文件
├── assets/       # 公共资源文件（CSS、JS、图片等）
└── README.md     # 本文件
```

- **pages/**：包含所有归档的 HTML 页面，每个文件为一个独立页面。
- **assets/**：存放页面引用的公共资源，如样式表、脚本、字体或图像文件。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独使用或引用。
- 资源文件路径相对固定，若需移动或部署，请注意更新相关引用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 如需部署到服务器，将整个仓库内容上传至 Web 目录即可。

## 维护说明

- 本仓库仅作为静态资源存储，不提供动态服务或后台接口。
- 页面内容更新时，请直接替换或新增文件，并保持目录结构清晰。
- 如无特殊说明，所有页面均为公开可访问。
